# Frontend Technical Plan

This skill provides the comprehensive frontend technical plan and UI/UX guidelines for the AI Visibility Analysis Platform.

## Core Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui or Radix UI primitives
- TanStack Query, TanStack Table
- Zustand or URL state

## Architecture & Data Models

The plan covers the essential MVP architecture, including Overview, Prompts, AI Chats, Citations, Competitors, and Knowledge Base modules. It also details the required data models (`Project`, `Prompt`, `AiAnswer`, `CitationSource`, `MetricSnapshot`).
